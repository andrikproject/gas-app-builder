⚡ GAS App Builder

Google Apps Script + Google Sheets — App Builder Extension

📖 Panduan Setup
1
Buka Google Sheets & Apps Script
Buka spreadsheet → Extensions → Apps Script
Hapus kode default, paste kode dari file Code.gs
2
Deploy Web App
Klik Deploy → New deployment
Pilih type: Web app
Execute as: Me
Who has access: Anyone
Klik Deploy → izinkan akses
3
Copy URL & Paste di Bawah
Copy URL Web App (https://script.google.com/.../exec)
Paste di input di bawah → klik Simpan & Tes

🔗 Konfigurasi Web App URL

🤖 Integrasi Gemini AI

📋 API Endpoints

GET ?action=getAll&sheet=NamaSheet → Semua data
GET ?action=getRow&sheet=NamaSheet&id=1 → Satu baris
POST {action:"addRow", sheet:"...", data:{...}} → Tambah data
POST {action:"updateRow", ...} → Update data
POST {action:"deleteRow", ...} → Hapus data
GET ?action=listSheets → Daftar sheet
GET ?action=stats&sheet=NamaSheet → Statistik
